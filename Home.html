HTML (HyperText Markup Language) is the foundation of web development, providing the basic structure for all websites. It defines the content of web pages using elements like headings, paragraphs, images, and links. Unlike programming languages, HTML is a markup language, meaning it does not perform logic or calculations but instead organizes and formats content for browsers to display. Every webpage you visit is built using HTML in some form, making it an essential skill for web developers and designers alike.

The tags I used are:

  • Heading tag
  • Bold tag
  • Para tag
  • Div tag
  • Table tag
  • Hyperlink tag
  • List tag
